# LieLieTea Bot
A MilkTea Flavored Chat Bot.

![GitHub](https://img.shields.io/github/license/yhluk/LLT-Bot)

##  简介
LLT-Bot (LieLieTea-Bot)，又名七筒 Chitung，是一个用于服务简体中文 Furry 社群的 QQ 机器人项目。
